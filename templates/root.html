These are front-ends to popular services, which you can use to improve your privacy, bypass restrictions, or reduce bandwidth usage.

If you have any feedback regarding these instances, you can send them here.

Why use my front-end instances over anyone elses
  • Automatic updates - these instances check for updates every 3 hours. This means that you will get the latest and greatest privacy experience.
  • EU based - my servers are located in France and Poland.
  • No logs - I will never figure out who is using my instances.
  • Security - my server does not have any unnecessary ports open and we use Caddy instead of Nginx which is a more secure web server. I also containerize all the services.
  • No modifications done to the frontends - just a nice, stock experience like the frontend's developer intended.
